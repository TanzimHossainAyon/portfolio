An Efficient Deep Learning Framework for Diabetic Retinopathy Classification using Generative Data Augmentation and Knowledge Distillation
Tasnimul Mohammad Fahim, Jawad Bin Alam, Sarika Bintey Sabur, Tanzim Hossain Ayon, Shuvo Datta, Azwad Aziz, Amitabha Chakrabarty
Engineered an integrated three-stage deep learning pipeline — a class-conditional Progressive Growing GAN for photorealistic synthetic retinal image generation, a heterogeneous dual-branch ensemble of EfficientNet-V2-S and Vision Transformer (ViT-B/16) fused via AUC-weighted averaging for state-of-the-art severity grading, and knowledge distillation compressing a 108M-parameter ensemble into a 3.5M-parameter MobileNetV2 for resource-constrained deployment.

Experience

Research Contributions

Five core contributions from undergraduate thesis research at BRAC University.

01
Generative Data Augmentation via Progressive Growing GAN
Designed a class-conditional PG-GAN synthesising 6,338 photorealistic retinal fundus images across five severity classes, achieving FID 23.4 — outperforming medical GAN benchmarks and boosting Severe NPDR sensitivity by +20.9%.
02
Heterogeneous Ensemble Learning for DR Classification
Developed a dual-branch ensemble combining EfficientNet-V2-S for local lesion texture and Vision Transformer (ViT-B/16) for global spatial reasoning, fused via adaptive AUC-weighted averaging — achieving 95.8% accuracy and 0.975 QWK on APTOS 2019.
03
Model Compression via Knowledge Distillation
Compressed a 108M-parameter ensemble teacher into a 3.5M-parameter MobileNetV2 student via knowledge distillation, achieving 96.8% parameter reduction and 98.8% FLOPs reduction while retaining 90.0% accuracy for mobile deployment.
04
Cross-Dataset Generalisation and Clinical Validation
Zero-shot validation on DDR (147 hospitals) and Messidor-2 achieved only 2.1–4.6% degradation versus the typical 10–15% in literature — confirming robust generalisation across diverse clinical environments.
05
Model Interpretability via Grad-CAM Visualisation
Systematic Grad-CAM analysis validated the model correctly focuses on clinically relevant pathologies — microaneurysms, haemorrhages, neovascularisation — addressing the black-box concern in medical AI.
